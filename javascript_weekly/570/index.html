    # The Best of JavaScript Weekly in 2021

    This is the last issue of the year (we're back on January 7, 2022) so we're taking a look back at some of the most popular items of 2021, starting with top links overall before moving on to top libraries, tools, videos, and more. You'll also find a quick month by month rundown in this issue, so there's sure to be something that'll catch your attention 😃
    ____
    Peter Cooper, your editor

    1. ES2021: Here's the Features (Quickly) (opens new window) — Back in June of this year, the Ecma General Assembly approved ES2021 (opens new window). Work continues with ECMAScript 2022 already having a draft spec (opens new window). But this post flying through what's new in ES2021 (not a lot!) was by far our most popular link of the year, amazingly. Dr. Axel Rauschmayer also had a popular post rounding up ES2021 (opens new window).

    ES2021：特性速递 — 早在今年 6 月，Ecma 大会就批准了 ES2021。 有关ECMAScript 2022 草案规范 (opens new window)工作仍在继续。但是这篇介绍 ES2021 新功能的帖子是我们今年最受欢迎的链接，挺令人惊讶的。 Axel Rauschmayer 博士还有一篇关于 ES2021 的热门帖子 (opens new window)。

    1. Comparing Svelte and React (opens new window) — After spending an extended period of time working with both, the author identified his preference based on first hand observations supported with real-world examples. I suspect 2022 will continue to see Svelte taking big leaps forward.

    比较 Svelte 和 React — 两个框架一起使用了很长一段时间后，作者根据真实示例确定了他喜欢哪个。我怀疑 2022 年 Svelte 将继续向前迈进一大步。

    (opens new window)

    Build a Mobile-Responsive Telehealth Pager App Using Stream’s Chat API (opens new window) — With Direct and Group Chats, emojis & Reactions, Built-in GIF support, the ability to edit & delete messages, specialized commands, and much more, this fully responsive Medical Pager is the best Chat App that you can currently find on YouTube.

    1. Comparing the New Generation of Build Tools (opens new window) — A detailed evaluation of a variety of approaches for bundling JavaScript and other front-end assets, including Snowpack, esbuild, Vite, and wmr. A handy overview.

    新一代构建工具对比 — 新一代构建工具对比，包括 Snowpack、esbuild、Vite 和 wmr。

    1. 'Rust is the Future of JavaScript Infrastructure' (opens new window) — It’s an opinion but the (still) growing evidence is compelling. Rust (as well as Go) is now being heavily used to replace parts of the JavaScript tooling ecosystem that might have otherwise been written in JS before (e.g. Rome, SWC, dprint).

    2. The State of Web Workers in 2021 (opens new window) — The web is single-threaded, but two years ago Surma told us why to always use Web Workers (opens new window) and why to bring multithreading to your solutions despite the difficulties. Almost every client supports (opens new window) them now, so Surma made the case again.

    3. Rethinking JavaScript's Ternary Operator (opens new window) — ternary ? "yes" : "no" … A lot of developers regard the ternary operator “with suspicion”, says James. Does its brevity warrant its use alone? Is it something you can “trust”? James also looks at a future alternative coming to the language.

    重新思考 JavaScript 的三元运算符—三元运算符 ? "true" : "false"……许多开发人员对三元运算符“持一点怀疑态度”，James 说：它的简洁性是否值得单独使用？它是你可以“信任”的东西吗？ James 还研究了该语言的未来替代方案。

    React Authentication, Simplified (opens new window) — In this article, we lay out a new approach to authentication (plus access control & SSO) in React applications.

    # 📅 JavaScript's 2021: Month by Month

    JANUARY: The State of JS 2020 results (opens new window) came out. As did Snowpack 3.0 (opens new window). React's original creator left Facebook (opens new window).

    FEBRUARY: npm 7.0 (opens new window) was released, as was Vite 2.0 (opens new window).

    MARCH: The Node.js team began thinking about the next 10 years of Node. (opens new window) Meanwhile, jQuery 3.6.0 was released. (opens new window) V8 sped up its release cycle. (opens new window) The ES2021 candidate spec (opens new window) was released. V8 9.0 (opens new window) was released. The Deno company was founded. (opens new window)

    APRIL: Node.js 16 (opens new window) was released. The ES module approach in Node began to take off. (opens new window)

    MAY: The Rome folks created a company, raised money, hired folks (opens new window) and more.

    JUNE: Undici 4 (opens new window) was released and pushed as a new better alternative HTTP client for Node.js. The plan for React 18 (opens new window) was unveiled. ES2021 was fully approved.

    JULY: Low code environment Node-RED 2.0 (opens new window) was released.

    AUGUST: The Wikimedia Foundation picked Vue.js (opens new window) as its frontend framework of choice. Vue 3.2 (opens new window) was released. The Ruby on Rails team considered a new approach to handling JavaScript (opens new window) in its popular Ruby framework.

    SEPTEMBER: Node.js gained an experimental package manager manager (sic) called Corepack (opens new window).

    OCTOBER: Node 17 (opens new window) was released. React's new docs site (opens new window) was unveiled. jQuery Mobile was deprecated. (opens new window) Parcel 2 (opens new window) was released. The Remix project took $3m in funding. (opens new window) Gatsby 4 (opens new window) was released.

    NOVEMBER: React 18 entered beta. (opens new window) Angular 13 (opens new window) was released. Remix was open sourced. (opens new window)

    DECEMBER: Create React App 5.0 (opens new window) was released.

    # 🛠 Top Code & Tools of 2021

    zx: A Tool for Writing Better Shell Scripts (opens new window) — Rather than using something like bash to throw together a quick script, zx provides a variety of niceties to do the same with the JavaScript you know and love.

    Over 100 Algorithms and Data Structures Demonstrated in JS (opens new window) — Examples of many common algorithms (e.g. bit manipulation, Pascal’s triangle, Hamming distance) and data structures (e.g. linked lists, tries, graphs) with explanations.

    Shortcut Puts the Agile in Agile and the “Can” in Kanban (opens new window)

    HTM 3.1.0: A JSX Alternative using Standard Tagged Templates (opens new window) — This clever library has been around a few years but continues to see updates. Think JSX-style syntax but in plain JavaScript (using tagged templates) that requires no transpilation but still supports things like rest spread and referencing components.

    (opens new window)

    Tagify: An Elegant Input Component for Tags (opens new window) — The polished demos show a lot of effort has been put in here. GitHub repo. (opens new window)

    Typehole: VS Code Tool to Automatically Generate TypeScript Types and Interfaces (opens new window) — An extension that helps you automate creating the initial static types for runtime values.

    Cheetah Grid: The Fastest Open-Source Data Table for the Web? (opens new window) — A bold claim from a library we hadn't encountered before. There’s a live demo here (opens new window) with 1,000,000 records in a table if you want to try it for yourself, though. A new release just dropped this week too.

    # 📺 Popular Videos of 2021

    ▶  JavaScript's Creator Brendan Eich Shares His JavaScript Story (opens new window) — Lex Fridman is an AI researcher who’s becoming well known for high quality interviews (opens new window) and here he spoke with Brendan Eich about his coding history, JavaScript’s full origin story, the browser wars, and Brave.

    ▶  What You Need to Know About React 18 (opens new window) — This year we shared this official plan for React 18 (opens new window) and while it's still 'almost here' the community set to work chewing over what it all means.

    Beyond onClick: Handling Events in React (opens new window)

    ▶  Coding Up a Mario Game in JavaScript with Kaboom.js (opens new window) — One of the most popular JavaScript YouTubers of 2021 showed us how to create a basic platformer in the course of an hour in her typically breezy, easy to follow manner.

    ▶  Ten JavaScript Clean Code Examples (opens new window) — Thoughts on ten examples of approaches worth considering if you’re trying to keep your code clean. It’s based on this article (opens new window) by Ibeh Ubachukwu.

    ▶  Have Single-Page Apps Ruined the Web? (opens new window) — The creator of frontend framework Svelte spoke about the criticisms lodged towards single-page apps and how ‘transitional apps’ may be the way forward. Might be an idea to keep in mind for 2022.
