    JavaScript Weekly #571

    2022/1/7

    (opens new window) 2021's JavaScript Rising Stars (opens new window) — Back for the 6th year running is this popular look at JavaScript projects that have continued to be popular on GitHub. Stars aren’t the only way to determine what projects are good, but it’s one way, and we tend to find it surfaces some worthwhile projects and libraries to check out. 2021 was a strong year for JavaScript and particularly for projects like Next.js, Vite, and zx!

    The ES2022 Edition of 'JavaScript for Impatient Programmers' (opens new window) — Dr. Axel has updated his popular book to ES2022 standards (ES2022 is to be finalized as a standard later this year) and you can still read (nearly) the whole book online (opens new window). Here’s the what’s new in ES2022 (opens new window) section.

    (opens new window)

    AI-Enabled Chatbots as Easy as “Hello World” (opens new window) — Build a conversational interface for your website or application with Botpress, the open-source platform designed for developers. Create production-ready, reliable, and scalable chatbots — without a team of data scientists or ML experts.

    Fuite: A Tool for Finding Memory Leaks in Web Apps (opens new window) — Given the choice of “load up DevTools and do lots of work in the Performance tab” vs “run this script”, the latter sounds like the easier option.. and Fuite aims to make the process of detecting memory leaks in your Web pages as easy as that.

    fuite 一个检测web应用内存泄漏的工具 — 分享一个用来检测内存泄漏的工具fuite

    The JS Self-Profiling API In Practice (opens new window) — It’s an API that allows webapps to control a sampling profiler for measuring client-side JavaScript execution times. Unsurprisingly it’s in Chrome 94+ only for now.

    Build a Monitoring App in 2 Hours (opens new window) — Free, live hands-on training on how to build an IoT, application, or infrastructure monitoring time series application.

    A Smarter JavaScript Mapper: array.flatMap() (opens new window) — You may be on the fence as to whether this approach is smarter or not, but it’s worth knowing.

    Is It Time for the JavaScript Temporal API? (opens new window) — “Date handling in JavaScript is a pain, it’s time for a proper Temporal API.”

    Introducing Metho: A Way to Add 'Superpowers' to JS (opens new window) — Every place I’ve seen this mentioned people have said “very clever, but please don’t use this in production code.” Take that as you will, but Metho certainly lets you twist JS in some rather nifty ways, but maybe you’d want to get your team onside first 😉 GitHub repo (opens new window).

    # 🛠 Code & Tools

    (opens new window)

    Perspective 1.1: Fast Streaming Data Visualization via WebAssembly (opens new window) — Originally built for J.P. Morgan, Perspective is for building real-time high performance interactive visualizations, powered by a C++ engine compiled to WASM under the hood. GitHub repo (opens new window) (with lots of examples (opens new window)).

    jsPDF 2.5: Client-Side JavaScript PDF Generation (opens new window) — Create tickets, documents, certificates, etc. all on the fly. There’s a live demo on the project’s homepage (opens new window).

    Tired of Egregious Egress? Try Vultr Instead (opens new window) — Instances start as low as $2.50 in select locations. Redeem your $100 credit to try Vultr for 14 days.

    Vanta.js: 3D WebGL Background Animations For Your Sites (opens new window) — Choose from one of the predefined background animations, then use the UI to tweak the settings. Certainly something you’d use sparingly, but a neat set of effects for when you need them.

    Solid 1.3: The VDOM-Less Declarative JS UI Library (opens new window) — Billed as a ‘big one’, this release focuses on improving Solid’s server side rendering story with support for HTML streaming and multiple isolated async hydration roots. Project homepage. (opens new window)

    Choices.js 10.0: A Configurable Select Box/Text Input Plugin (opens new window) — Lots of examples here, or you can go straight to the GitHub repo (opens new window).

    Try JavaScript Online: Simple Web-Based JavaScript Console Input (opens new window) — This is a console side-by-side with a web page, so you can run script snippets while learning. The default example web page is Eloquent JavaScript (opens new window) but you can enter any online resource.
