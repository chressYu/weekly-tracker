    # 什么是CSS自定义属性？

    CSS的自定义属性等同于JS中的let,它可以用来定义在样式中重复使用的属性。 同时他仍然遵守CSS的级联规则。 这就意味着我们可以给自定义属性定义默认属性值，然后根据需要来进行覆盖。

    可以在属性前面加两个破折号来实现自定义属性--my-property 在使用CSS函数 var() 进行调用
